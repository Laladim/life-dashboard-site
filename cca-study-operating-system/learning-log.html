Note-template chooser (which of Shela's 7 templates fits which lesson type)

Lesson typeBest templateWhy
A process / loop / sequence (agent loop, lookup-then-act, retry flow)Flow ChartThe thing IS a sequence; draw it as one.
Compare two options (plan vs direct, tool_choice auto/any/forced, project vs user config)Two ColumnExam loves "which one, when" - columns force the contrast.
One discrete concept w/ parts (MCP tool vs resource vs prompt, a single tool's anatomy)Box MethodDefinition + features + examples + when-to-use in one frame.
A landscape / overview lesson (intro to X, the 5 domains)Mind MapCaptures breadth + relationships, not depth.
A hierarchy of sub-topics (a domain broken into task areas)OutlineMirrors the nested structure.
Concept-heavy lesson where you'll have questionsCornellCues/questions column = built-in active recall (the workhorse default).
A dense list of independent factsSentenceFast capture, one fact per line.

Default to Cornell when unsure - its cue column doubles as a self-quiz.
